# Installation

Make sure you have already [downloaded](download.md) the plugin.

## Install the Plugin

1. Stop your server.
2. Place the `SpawnerSilk-X.X.X.jar` file into your `plugins/` folder.
3. Start the server.

If the plugin loads correctly, you should see a message similar to:

```text
[SpawnerSilk] Enabling SpawnerSilk vX.X.X
```

## After the First Start

- A default `config.yml` will be generated in the plugin data folder.
- The default localization file will be created in `plugins/SpawnerSilk/localization/`.
- You can reload configuration changes later with `/sps reload`.

If the plugin fails to start, check the server version, Java version, and console logs first. If needed, ask for help on Discord.
