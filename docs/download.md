# Download

You can download SpawnerSilk from the official project pages below:

- [BukkitDev](https://dev.bukkit.org/projects/spawnersilk)
- [CurseForge](https://www.curseforge.com/minecraft/bukkit-plugins/spawnersilk)
- [GitHub](https://github.com/Stellionix/SpawnerSilk) for source code

!!! warning
    If you find SpawnerSilk on another platform, avoid downloading it there unless you trust the source.

## Requirements

SpawnerSilk supports Minecraft `1.12+` and runs on Java `8+`.

Supported server software includes:

- Bukkit
- Spigot
- Paper
- Purpur
- Tuinity
- Most Bukkit/Spigot-compatible forks
